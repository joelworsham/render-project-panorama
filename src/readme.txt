== Description ==

Project Panorama - "This WordPress project management plugin can help keep your clients happy and reduce your stress by visually communicating project progress."

Project Panorama is awesome and comes with a few useful shortcodes. This integration with Render allows you to use them from right within the Render window! Bring you convenience and ease when using these shortcodes.

== Installation ==

Using this plugin is very simple. All you have to do is:

1. Upload the `render-psp` folder to the `/wp-content/plugins/` directory

1. (b) OR install the zip file through the "Upload Plugin" installer.

2. Activate the plugin through the 'Plugins' menu in WordPress
